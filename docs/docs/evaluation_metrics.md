## Metrics

### 1. Grounding Accuracy
- 1 = uses real user numbers (income, category amounts, percentages)
- 0 = generic (no real numbers)

### 2. Quiz Relevance
- 1 = quiz is clearly based on the user’s input (categories, spending, etc.)
- 0 = generic or unrelated

### 3. Tip-to-Rule Mapping
- 1 = tip clearly matches a rule from financial_rules.md
- 0 = vague or not traceable
